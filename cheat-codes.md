# Phil's Escape — Cheat Codes

This document lists all available cheat codes in Phil's Escape. Cheat codes can only be activated during gameplay (not in menus or paused state).

---

## World Skip Cheat

**What it does:** Advances to the next world (or next level if already on the final level). Resets your current level but preserves your score, lives, and health.

| Platform | Input |
|----------|-------|
| **Keyboard** | `X` `A` `V` `I` `E` `R` (type quickly) |
| **Mobile/Touch** | ▲ ▲ 🛡 ▲ ▲ 🛡 ▲ ▲ 🛡 |

**Notes:**
- Each button press must be entered within **1.4 seconds** of the previous one, or the code resets
- Cannot be used during the final boss fight
- Text popup will show `WORLD X`, `LEVEL X`, or `FINAL BOSS` when activated
- For mobile: Jump button is ▲, Shield button is 🛡

## Usage Tips

- **Timing matters:** All button presses must happen quickly (within 1.4 seconds total) or the cheat resets
- **During gameplay only:** These cheats only work when you're actively playing (not in start screen, pause menu, or level select)
- **Not during boss:** You cannot activate cheats during the final boss encounter
- **Visual feedback:** Look for the colored text popup confirming which world/level you're advancing to
